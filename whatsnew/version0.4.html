Release notes for nitime version 0.4¶

Summary of changes¶

Major changes introduced in version 0.4 of nitime:

  1. LazyImports: Imports of modules

    are delayed until they are actually used. Work led by Paul Ivanov

  2. TimeArray math: Mathematical

    operations such as multiplication/division, as well as min/max/mean/sum are now implemented for the TimeArray class. Work led by Paul Ivanov.

  3. Replace numpy FFT with scipy FFT. This should improve performance. Work

    instigated and led by Alex Gramfort.

  4. Scipy > 0.10 compatibility: Changes to recent versions of scipy have caused

    import of some modules of nitime to break. This version should have fixed this issue.
